# Quiz Agro Sustentável

## Sobre o projeto

O **Quiz Agro Sustentável** é um jogo de perguntas e respostas sobre agricultura, pecuária, meio ambiente e sustentabilidade.

O jogador precisa responder 10 perguntas antes que o tempo acabe.

Tema do projeto:

**Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente**

## Funcionalidades

- 10 perguntas.
- 3 alternativas por pergunta.
- Alternativas misturadas.
- Cronômetro de 60 segundos.
- Pontuação durante o jogo.
- Mensagem final conforme o desempenho.
- Layout simples e responsivo.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## Como usar

1. Baixe o projeto.
2. Abra a pasta no VS Code.
3. Abra o arquivo `index.html` no navegador.
4. Responda às perguntas antes que o tempo acabe.
